Verify SHA-256 Checksums

Download the file first; then compute its SHA-256 hash and compare it to checksums.txt. If the hashes match exactly, your download is intact.

Windows PowerShell
cd $HOME\Downloads
Get-FileHash .\SecureVault-v0.1.1-Windows.exe -Algorithm SHA256
macOS (Terminal)
cd ~/Downloads
shasum -a 256 SecureVault-v0.1.1-macOS
Linux / WSL (Terminal)
cd ~/Downloads
sha256sum SecureVault-v0.1.1-Linux
